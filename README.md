# Задача - разработать SSE сервис на Java

## Критерии приемки:
- [ ] Сервер должен открывать streaming на http://localhost:4000/sse
- [ ] При коннекте клиента сервер шлет сообщение ```CONNECTED```
- [ ] Добавить эндпонит ```POST /trigger-message``` при вызове которого произойдет отправка сообщения на клиент (любое сообщение)
- [ ] Сервис завернут в docker, команду запуска положить в файл ```start.sh```

## Дополнительно задание
- [ ] При подключении нескольких клиентов, каждому раздать уникальный ID и присылать сообщение ```CLIENT {ID} CONNECTED```
